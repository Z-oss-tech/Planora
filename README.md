Planora is a modern productivity app designed to help users manage tasks, track habits, and stay organized with a clean and aesthetic UI.

---

## ✨ Features

* 📝 Create, update, and delete tasks
* 📌 Add subtasks and priorities
* 🔔 Set reminders
* 🔁 Recurring tasks support
* 📊 Habit tracking
* 🎨 Premium UI with glassmorphism design
